📊 КЛЮЧЕВЫЕ ФАЙЛЫ ДЛЯ ИЗУЧЕНИЯ КОДА:

Начинающим:
- game/snake.py      → Простая логика змейки
- game/food.py       → Генерация еды
- game/config.py     → Настройки (легко менять)

Продвинутым:
- ml/analyzer.py     → Алгоритм A*, метрики
- ml/ai_player.py    → AI агент
- game/game.py       → Сбор данных

Визуализация:
- ml/visualizer.py   → GUI для анализа
- ml/ai_demo.py      → Демо режим
